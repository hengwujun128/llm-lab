RecursiveCharacterTextSplitter 分割逻辑

配置: separators = ['。', '！', '？']  /  chunkSize = 500  /  chunkOverlap = 50

算法流程图

核心是递归 — 当片段超过 chunkSize，用下一个分隔符继续切割

分步示例（点击数字切换）

mergeSplits 合并逻辑

遍历 splits对每个 split 计算 total + len + 分隔符长度
total + len > chunkSize?是 → 输出当前 chunk，保留 overlap 部分
保留 overlap从头部移除 splits 直到 total <= chunkOverlap
继续添加把当前 split 加入 currentDoc，total += len
最后一批循环结束后，剩余 currentDoc 合并为最后一个 chunk

Chunk 重叠示意

注意
当前 separators 只有 ['。', '！', '？']。如果文本中不含这些标点，将无法继续细分。 建议添加 '，'、' '、'' 作为兜底，其中 '' 会按单字符拆分，确保不超过 chunkSize。
